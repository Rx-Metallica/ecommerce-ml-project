# 🛒 E-Commerce Customer Segmentation & Satisfaction Prediction

## 📌 Project Overview
An end-to-end Machine Learning project that predicts customer 
satisfaction and segments customers using a real e-commerce dataset.

## 📊 Dataset
- Source: Kaggle — E-Commerce Customer Behavior Dataset
- 348 customers | 11 features

## 🔧 Steps Covered
- Exploratory Data Analysis (EDA)
- Data Preprocessing & Encoding
- Train/Test Split
- Random Forest Classifier
- Parameter Tuning (GridSearchCV)
- 5-Fold Cross Validation
- AUC-ROC Score
- Feature Importance
- K-Means Customer Segmentation
- Business Insights
